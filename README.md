## ğŸš¦ Getting Started

### Prerequisites
- Node.js 16.14 or higher
- npm or yarn
- Python 3.8+

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ai-driven-development/AI_Robotics_book.git
cd AI_Robotics_book
```

2. Install dependencies:
```bash
cd docs
npm install
```

3. Start the development server:
```bash
npm run start
```

4. Open your browser and navigate to `http://localhost:3000`
